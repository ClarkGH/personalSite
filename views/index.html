About

Before becoming the badass fullstack developer he is today, Clark has worked as a recreational therapist, a customer service rep, a fast food manager, and a stadium supervisor.

Projects

Meet New World Radio. This project was created to play music from any major country from any year between 1940-2014 and be reminiscent of the old black and white iPod. While the project is now deprecated after changes to the source APIs, you can still view the source code here.

This project was built using Ruby, Rails, RSpec, Capybara, Javascript, JQuery, PostgreSQL, and the Youtube and Echonest APIs.
